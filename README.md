A lightweight utility tool that generates valid (fake) South African ID numbers for testing and development purposes. Deployed as a developer tool within the **Kirov Dynamics** toolkit.

## ✨ Features

- **🆔 Valid Format Generation**: Produces structurally valid 13-digit SA ID numbers.
- **🎂 Date of Birth Encoding**: ID contains embedded date of birth.
- **⚡ Instant Output**: Zero dependencies, runs in the browser.
- **🔒 Privacy Safe**: Generates fake, non-real numbers for dev/test use only.
